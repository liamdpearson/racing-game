Real-time local multiplayer racing game using threading and sockets. You can play by yourself or with up to 5 people total. You and your friends must be on the same Wi-Fi network to play together.

Written in Python using the Arcade library version 2.6.17.

## Features:
1. Drifting
2. Speed Boosts
3. Slow Spots
4. Powerups
5. Coins
6. SFX
7. Multiple Maps
8. Multiple Cars
9. Other Player location prediction
10. Robust server
11. Customizable Keybinds
