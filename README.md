# Speed Racing
![Python](https://img.shields.io/badge/python-3.11.9-blue)
![Python](https://img.shields.io/badge/arcade-2.6.17-red)

Real-time local multiplayer racing game using threading and sockets.

Written in Python using the Arcade library version 2.6.17.

Demo here:

## Features:
1. Drifting
2. Speed Boosts
3. Slow Spots
4. Powerups
5. Coins
6. SFX
7. Multiple Maps
8. Multiple Cars
9. Other Player location prediction
10. Robust server
11. Customizable Keybinds
